Tips

  • Chat: Ask specific questions; sessions remember history; wiki entries searched first
  • Documents: Files must be on this computer; any plain text; duplicates auto-detected
  • Wiki: Improves answer quality; re-compile after adding new docs
  • Settings: GPU layers 0=CPU, -1=all; larger context = more memory; lower temp = focused
